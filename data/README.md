# Data Directory

This directory is intentionally kept out of Git for large market data files.

Expected local layout:

```text
data/raw/01302020.NASDAQ_ITCH50.bin      # full ITCH BinaryFILE, downloaded locally
data/filtered/test.bin                   # mini feed generated by itch/build/filter
```

Both `data/raw/` and `data/filtered/` are ignored by Git except for `.gitkeep` placeholders.
